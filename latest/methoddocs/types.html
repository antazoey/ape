A serializable type representing the type of a contract. For example, if you define your contract as contract MyContract (in Solidity), then MyContract would be the type.

property calls¶

The call-methods (read-only method, non-payable methods) defined in a smart contract.

Returns

list[ABI]

property constructor¶

The constructor of the contract, if it has one. For example, your smart-contract (in Solidity) may define a constructor() public {}. This property contains information about the parameters needed to initialize a contract.

property events¶

The events defined in a smart contract.

Returns

list[ABI]

property fallback¶

The fallback method of the contract, if it has one. A fallback method is external, has no name, arguments, or return value, and gets invoked when the user attempts to call a method that does not exist.

property transactions¶

The transaction-methods (stateful or payable methods) defined in a smart contract.

Returns

list[ABI]

class ape.types.contract.Source(_keep_fields_: Set[str] = {}, _skip_fields_: Set[str] = {}, checksum: Optional[ape.types.contract.Checksum] = None, urls: List[str] = [], content: Optional[str] = None, installPath: Optional[str] = None, type: Optional[str] = None, license: Optional[str] = None, **kwargs)¶

Bases: ape.types.abstract.SerializableType

A serializable type representing source material.

checksum¶

The checksum of the source. Set by calling the method compute_checksum().

compute_checksum(algorithm: str = 'md5', force: bool = False)¶

Compute the checksum if content exists but checksum doesn’t exist yet. Or compute the checksum regardless if force is True.

Raises

ValueError – When the content is not yet loaded.

Parameters

  • algorithm (str, optional) – The algorithm to use when computing the checksum. Defaults to "md5".

  • force (bool, optional) – Set to True to force a re-check. Defaults to False.

license¶

The license of the source, if one exists, such as the MIT license.

type¶

The type of source, such as "interface".

urls¶

The URLs by which to obtain the source, such as IPFS URLs.
